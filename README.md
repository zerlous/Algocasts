### Javascript

####  Run
```bash
$ git clone https://github.com/zerlous/Algocasts.git
$ cd Algocasts/javascript
$ npm install	/ yarn install
$ npm run test / yann run test
```

#### Create Problem

```bash
$ node mkdir.js -n [leetcode number] -s [solution name]
eg:
$ node mkdir.js -n 1 -s twoSum
```
